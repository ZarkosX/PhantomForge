## What It Does
Clones **everything** from one server to another using your personal Discord token:
- Server name + icon
- All channels (text/voice/categories) + order
- Roles + colors + permissions
- Emojis
- Exact permission overwrites

---

## How to Use
1. ```
   https://github.com/ZarkosX/PhantomForge.git
   cd PhantomForge
   ```
2. Install:
   ```
   pip install discord.py aiohttp
   ```
3. Run:
   ```
   python PhantomForge.py
   ```
4. Enter:
   - Your **Discord token** (from browser dev tools)
   - **Source Server ID** (copy from)
   - **Target Server ID** (paste into)
